This is a [Next.js](https://nextjs.org) project bootstrapped with [`create-next-app`](https://nextjs.org/docs/app/api-reference/cli/create-next-app).

## How to use

Must have a .env.local file in root with:

```bash
STARLING_ACCOUNT_UID
STARLING_ACCESS_TOKEN
STARLING_CATEGORY_UID
```

Make sure sandbox user has not expired and has correct access token.

Run this in the terminal at root:
```bash
npm run dev
```
